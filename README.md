"# react-redux-react-router4" 

如何开始？
首先你得先clone下来，
一、安装依赖，环境是node.js 直接cnpm install  --------那么依赖安装完成---------；
二、npm run dev ----------------启动服务--------；
三、打开浏览器在地址栏中输入127.0.0.1:4000,恭喜你已经可以看到界面啦：----其实这个端口号在webpack.config.js中可以看到啦\(≧▽≦)/！---；
四、登陆：username:admin,password:admin; 这里就是这个后台管理系统的主要页面框架啦。
